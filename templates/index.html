How to use & Information

Information

This tool is intended to be used by teachers to make creating online tests easier.

How to use

To use this tool, you first put in the name of the test you are creating.
Next, you input the question, as well as it's 4 possible answers (multiple choice).
After this, click submit! Your test will be uploaded to the server for students to take, and a link will be provided for the students to take the test.

How it works

It works by using a Python web-server (using the open-source Flask framework), and an HTML/CSS/JavaScript website.
The data inputted by the user is sent to the server for storage, where it can be referenced whenever needed.
